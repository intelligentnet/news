# INPUT
A json object with a story title and many chapters names and summarized chapter contents is supplied. 
# CHAPTERS CONTENTS
Take each chapter summary.
Develop the characters from the summary.
Develop the themes in the summary.
Create a longer chapter story humanized text of over 1000 words to replace the summary.
# OUTPUT 
The title is an HTML heading.
Followed by an HTML table with exactly the same chapter names as before but replace the chapter summary with the longer chapter story text.
Return everything as HTML.
