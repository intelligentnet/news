# STORY
Tell a story which has a Title and ${chapters} chapters each with a Number, Name and Summary field.
# FORMAT
The title is a text field with name and id of title and the value is generated.
The chapters are laid out in a table in the form.
Each Name text field has id and name parameters called name_NN and a generated value.
Each Summary textarea has rows=4 and cols=150 and has id and name parameters called summary_NN and a generated value.
Humanize all of the text using simple British English.
Add a submit button and submit to /taledetail.
Return only HTML without any comments.
